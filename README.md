# 📚 Интеллектуальная система озвучивания аудиокниг с алгоритмическим улучшением просодии

## Описание проекта
Система синтеза речи для художественных текстов с улучшенной просодией.
Проект реализует многоступенчатый пайплайн обработки:
1. **NLP-анализ**: морфологический разбор, расстановка пауз на основе синтаксиса
2. **TTS-синтез**: предобученная модель Silero TTS v5
3. **Пост-обработка**: нормализация аудио, конкатенация чанков

## Научная ценность
- Улучшение качества синтеза **без переобучения модели** за счёт алгоритмической предобработки текста
- Экспериментальное обоснование параметров: sample_rate, размер чанка, длительность пауз
- Модульная архитектура для масштабирования и замены компонентов

## Технологии
- **Язык**: Python 3.12
- **TTS**: Silero TTS v5 (PyTorch, torch.hub)
- **NLP**: pymorphy3 (морфологический анализ русского языка)
- **Audio**: soundfile, numpy, librosa
- **UI**: Gradio (веб-интерфейс)
