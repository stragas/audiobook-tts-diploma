# Архитектура пайплайна синтеза аудиокниг

## Компоненты системы

| Компонент | Модуль | Назначение |
|-----------|--------|------------|
| NLP-препроцессинг | text_processor.py | Расстановка пауз, анализ текста |
| TTS-движок | Silero TTS v5 | Синтез речи из текста |
| Пост-обработка | tts_pipeline.py | Нормализация, конкатенация |
| Интерфейс | gradio_app.py | Веб-сервис для пользователей |

## Научная ценность

1. **Улучшение просодии без переобучения**
   - Алгоритмическая расстановка пауз на основе синтаксиса
   - Экспериментальное обоснование длительности пауз

2. **Многоступенчатая обработка**
   - Текст → NLP → Чанки → TTS → Пост-обработка → Аудио
   - Каждый этап тестируется отдельно

3. **Модульная архитектура**
   - Легко заменить TTS-движок (Silero → Coqui → VITS)
   - Легко добавить новые функции (SFX, дикторы)

## Экспериментальное обоснование параметров

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| sample_rate | 16000 Гц | Эксперимент с 4 вариантами (8k/16k/24k/48k) |
| max_chars | 500 | Лимит модели Silero + тесты на стабильность |
| pause_short | 0.3 сек | Субъективная оценка естественности |
| pause_medium | 0.6 сек | Субъективная оценка естественности |
| pause_long | 1.0 сек | Конец предложения |
| pause_xlong | 1.5 сек | Абзац, многоточие |
