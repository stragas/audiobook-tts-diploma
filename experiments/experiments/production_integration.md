# Production-интеграция: Веб-сервис синтеза аудиокниг

## Требование диплома
«Интегрировать в production: реализовать работающее приложение 
(веб-сервис, Telegram-бот, API-сервис)»

## Реализация
Веб-интерфейс на базе Gradio с полным пайплайном обработки:
1. Приём текста от пользователя
2. NLP-предобработка (расстановка пауз)
3. TTS-синтез (Silero v5)
4. Пост-обработка (нормализация)
5. Выдача аудиофайла

## Архитектура
| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Frontend | Gradio | Веб-интерфейс |
| Backend | Python 3.12 | Логика пайплайна |
| NLP | pymorphy3 | Морфологический анализ |
| TTS | Silero TTS v5 | Синтез речи |
| Audio | soundfile, numpy | Обработка аудио |

## Экспериментальное обоснование параметров

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| max_chars | 140 | Предупреждение Silero >140 символов |
| sample_rate | 16000 | Эксперимент с 4 вариантами (8k/16k/24k/48k) |
| max_text | 10000 | Ограничение для стабильности веб-сервиса |
