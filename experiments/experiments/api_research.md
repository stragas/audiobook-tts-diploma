# Экспериментальное обоснование параметров Silero TTS v5

## Цель
Определить корректные параметры интеграции Silero TTS для синтеза 
русскоязычной речи в рамках проекта «Нейронная сеть для озвучивания аудиокниг».

## Методология
1. Программный анализ сигнатур методов (inspect.signature)
2. Эмпирическое тестирование различных sample_rate
3. Исследование формата возвращаемых данных
4. Субъективная оценка качества воспроизведения

## Эксперимент 1: Выбор sample_rate

| Sample Rate | Длительность | Скорость восприятия | Оценка |
|-------------|--------------|---------------------|--------|
| 48000 Гц | 1.40 сек | 3x быстрее нормы | ❌ Неразборчиво |
| 24000 Гц | 2.80 сек | 1.5x быстрее нормы | ❌ Быстро |
| **16000 Гц** | **4.21 сек** | **Нормальная речь** | ✅ **Принято** |
| 8000 Гц | 8.42 сек | 2x медленнее нормы | ❌ Заторможено |

**Вывод:** sample_rate = 16000 Гц обеспечивает естественную скорость речи.
